Descrição: Este repositório foi criado com o objetivo de praticar o versionamento de pastas usando o Git e o GitHub. Além disso, o conteúdo utilizado foram exercícios de lógica com pseudocódigo "Portugol".



Ciclo de clonagem do repositório:

* copiar a URL do repositório no git hub
* abrir o terminal e entrar na pasta que a clonagem será feita
* git clone \[URL do repositório]



Ciclo de Versionamento:

* git add "nomeDoArquivo"
* git commit -m "Mensagem para etiquetar esse pacote")
* git push
